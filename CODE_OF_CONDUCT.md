# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for all.

Our standards:
- Be respectful and considerate.
- Assume good intent and communicate with care.
- Harassment, discrimination, and abusive behavior are not tolerated.
- Respect differing viewpoints and experiences.

Scope:
This code of conduct applies to all project spaces, including issues, pull requests, discussions, and real-time communication channels.

Enforcement:
To report a concern, open a GitHub issue and request a private channel, or use GitHub's reporting tools. Maintainers will review and respond as promptly as possible.

Attribution:
This policy is inspired by common open-source codes of conduct and adapted for this project.
