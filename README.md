# Fennec

Fennec is a new open-source server control plane (controller + agent + UI) designed for modern hosting operations.

Status: bootstrap only (docs, repo structure, CI scaffolding).

Start here:
- docs/manifesto.md
- docs/spec.md

Key directories:
- controller/
- agent/
- ui/
- pkg/
- tools/
- docs/

Migration tooling lives in a separate repo: `fennec-migrate`.
